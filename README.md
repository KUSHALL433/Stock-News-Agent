# Stock-News-Agent
A Langgraph based flow that gathers news articles from moneycontrol.com and analyses them to generate a summary report.It implements parallel processing of articles from all the sources for faster processing.
<br>
This app is now live at [Stock AI News Summary](https://stocknewssummary.streamlit.app/)
